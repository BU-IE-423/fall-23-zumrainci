Introduction¶

This project requires us to discover the concept of pair trading and to use our knowledge of control charts in order to simulate a decision making system. This simulation gives the time periods of when to buy or sell the concerning stocks. First, we decided on the time range of the simulation. We simulated the first three months of 2023 by using the data between 2021-2022. Then, we found the most correlated stock pairs among the data. After that, we constructed the control charts by 2 sigma value in order to find the control limits.

def combine_csv(files):
    all_data = pd.DataFrame()
    for file in files:
        df = pd.read_csv(file, parse_dates=['timestamp'], index_col='timestamp')
        all_data = pd.concat([all_data, df])
    all_data.sort_index(inplace=True)  # Ensure the data is sorted by time
    all_data.interpolate(method='linear', inplace=True)  # Interpolate missing values
    return all_data

We construct a function that can fit a linear regression model. In this case, X is the independent variable, y is the dependent variable. So, the model fitting a line that best predicts the price of y (in first pair SAHOL), based on the price of X (in this case, KCHOL). Then we calculate the residuals, with def calculate_residuals function. The residuals calculated in the function are the differences between the actual observed prices of 'SAHOL' and the prices predicted by the model from 'KCHOL' prices within the specified range.

def linear_regression_model(df, stock1, stock2):
    X = df[df['short_name'] == stock1]['price'].values.reshape(-1, 1)
    y = df[df['short_name'] == stock2]['price'].values
    model = LinearRegression().fit(X, y)
    return model

model = linear_regression_model(stock_data, stock1, stock2)

def calculate_residuals(df, model, start_date, end_date):
    filtered_df = df[(df.index >= start_date) & (df.index <= end_date)]
    X = filtered_df[filtered_df['short_name'] == stock1]['price'].values.reshape(-1, 1)
    y_actual = filtered_df[filtered_df['short_name'] == stock2]['price'].values
    y_predicted = model.predict(X)
    residuals = y_actual - y_predicted
    return residuals, filtered_df

We identified the initial capital 100,000 TL. When a buy signal occurs, we use all the capital to buy that stock until a sell signal occurs. Then, the profit or loss is calculated by final_portfolio_value - initial_capital at the end of simulation time.

In [42]:
# Function to simulate the trading based on signals
def simulate_trading(signals, initial_capital):
    capital = initial_capital
    sahol_shares = 0
    kchol_shares = 0
    current_stock = None  # Track which stock is currently being held

    for signal in signals:
        date, kchol_price, kchol_decision, sahol_price, sahol_decision = signal

        if current_stock is None:
            # First signal: decide which stock to buy initially
            if sahol_decision == 'BUY':
                sahol_shares = capital // sahol_price
                capital -= sahol_shares * sahol_price
                current_stock = 'SAHOL'
            elif kchol_decision == 'BUY':
                kchol_shares = capital // kchol_price
                capital -= kchol_shares * kchol_price
                current_stock = 'KCHOL'

        else:
            # Subsequent signals: execute trades based on the current stock
            if current_stock == 'SAHOL' and kchol_decision == 'BUY':
                # Sell SAHOL and buy KCHOL
                capital += sahol_shares * sahol_price
                sahol_shares = 0
                kchol_shares = capital // kchol_price
                capital -= kchol_shares * kchol_price
                current_stock = 'KCHOL'

            elif current_stock == 'KCHOL' and sahol_decision == 'BUY':
                # Sell KCHOL and buy SAHOL
                capital += kchol_shares * kchol_price
                kchol_shares = 0
                sahol_shares = capital // sahol_price
                capital -= sahol_shares * sahol_price
                current_stock = 'SAHOL'

    # Calculate final value of the portfolio
    final_portfolio_value = capital + sahol_shares * sahol_price + kchol_shares * kchol_price
    profit_or_loss = final_portfolio_value - initial_capital
    return profit_or_loss

# Perform time series analysis 
def time_series_analysis(df, stock1, stock2):
    X = df[df['short_name'] == stock1]['price'].values.reshape(-1, 1)
    y = df[df['short_name'] == stock2]['price'].values

    model = LinearRegression().fit(X, y)

    residuals = y - model.predict(X)

    # Time series modeling (ARIMA)
    ts_model = sm.tsa.ARIMA(residuals, order=(1, 0, 1)).fit()

    forecasted_residuals = ts_model.predict(start=len(residuals), end=len(residuals) + len(X) - 1)

    mean_residual = np.mean(residuals)
    std_residual = np.std(residuals)
    updated_control_limits = (mean_residual + sigma_mult * std_residual, mean_residual - sigma_mult * std_residual)

    return model, updated_control_limits

def simulate_trading(signals, initial_capital):
    capital = initial_capital
    sahol_shares = 0
    kchol_shares = 0
    current_stock = None

    for signal in signals:
        date, kchol_price, kchol_decision, sahol_price, sahol_decision = signal

        if current_stock is None:
            if sahol_decision == 'BUY':
                sahol_shares = capital // sahol_price
                capital -= sahol_shares * sahol_price
                current_stock = 'SAHOL'
            elif kchol_decision == 'BUY':
                kchol_shares = capital // kchol_price
                capital -= kchol_shares * kchol_price
                current_stock = 'KCHOL'
        else:
            if current_stock == 'SAHOL' and kchol_decision == 'BUY':
                capital += sahol_shares * sahol_price
                sahol_shares = 0
                kchol_shares = capital // kchol_price
                capital -= kchol_shares * kchol_price
                current_stock = 'KCHOL'
            elif current_stock == 'KCHOL' and sahol_decision == 'BUY':
                capital += kchol_shares * kchol_price
                kchol_shares = 0
                sahol_shares = capital // sahol_price
                capital -= sahol_shares * sahol_price
                current_stock = 'SAHOL'

    final_portfolio_value = capital + sahol_shares * sahol_price + kchol_shares * kchol_price
    profit_or_loss = final_portfolio_value - initial_capital
    return profit_or_loss

Discussion¶

Task 1 is easy to understand and to implement. We use linear regression for task 1 which relies on the relationship between two stocks and use residuals to identify the trading opportunities. Since this method is simpler, it is faster to compute. So, we can use this method for a long time duration, which we have more data, and for the situations where computational resources are limited. If the relationship between the pair of stocks is stable over time, this strategy can be quite effective. However, this method assumes that the variance of the residuals is constant over time, which may not hold always. Also, financial markets are complex, and a simple linear relationship might not capture all the dynamics between the paired stocks. It does not account for changes in the relationship between the stocks over time.

On the other hand using time series analysis allows the model to adapt to changes over time. So, it provides more accurate predictions in markets where relationships evolve. Time series models, like ARIMA as we used, can handle non-stationary data, making them more robust in certain market conditions. But, it's more complex to implement and requires more computational resources. This complexity can make the model harder to understand. Effective time series analysis requires a substantial amount of historical data for accurate modeling, which might not be available for all stock pairs.

Finally, the basic strategy is suitable for less volatile markets where the pair's price relationship is stable and linear over time. It's ideal for traders looking for a straightforward, easy-to-implement strategy. On the other hand, the advanced strategy is better suited for markets where stock relationships are more dynamic and complex.

Conclusion¶

Throughout this assignment, we explored two distinct approaches to pairs trading strategies: a basic strategy using constant variance assumption and an advanced strategy incorporating time series analysis. The first way was straightforward and used linear regression. It was quick and easy to understand. However, it assumed that the market doesn't change much, which isn't always true. The second method was more complicated and used time series analysis. It seemed better for dealing with markets that change a lot, but it was also more complex and needed more data. Both methods have their effectiveness and limitations, so it's important to pick the right one depending on pairs and how much risk the trader is willing to take.
